# ระบบวิเคราะห์หนี้คงค้าง

ระบบวิเคราะห์หนี้คงค้างลูกค้าสำหรับธุรกิจ พร้อมการแจ้งเตือนและสรุปผลตามสาขา

## การติดตั้ง

1. Deploy Google Apps Script (ดูใน Code.gs)
2. แก้ไข GAS_URL ใน index.html
3. เปิด GitHub Pages

## ฟังก์ชันหลัก
- แสดงสรุปสถิติหนี้
- การแจ้งเตือนลูกค้าที่มียอดค้าง
- สรุปตามสาขา
- ค้นหาและจัดเรียงข้อมูล
- Export ข้อมูล

## เทคโนโลยี
- HTML5/CSS3
- JavaScript/jQuery
- Bootstrap 5
- DataTables
- Google Apps Script API
